# 中国人的人情世故知识体系

下面是一套偏“社会运行逻辑”的梳理。

```text
中国人的人情世故知识体系
├─ 1. 底层逻辑
│  ├─ 关系本位：很多事先看“人与人的关系”，再看规则本身
│  ├─ 情理法排序：私下常讲“情、理、法”，正式场合强调“法、理、情”
│  ├─ 面子机制：尊严、体面、身份、台阶，是互动中的重要资源
│  ├─ 互惠原则：来而不往非礼也，帮助、礼物、人情常形成往来账
│  ├─ 分寸感：什么话能说、说到几分、何时说，比内容本身更重要
│  └─ 场合意识：同一句话，在饭局、会议、家里、微信里含义可能不同
│
├─ 2. 核心概念
│  ├─ 人情
│  │  ├─ 含义：情分、帮忙、照顾、往来、亏欠感
│  │  ├─ 特点：可积累、可消耗、可转移、可被记住
│  │  └─ 风险：欠太多人情会失去主动权
│  ├─ 面子
│  │  ├─ 给面子：公开尊重、留余地、适当抬举
│  │  ├─ 留面子：批评不当众撕破，拒绝不直接羞辱
│  │  └─ 丢面子：公开否定、揭短、让人下不来台
│  ├─ 关系
│  │  ├─ 熟人关系：讲情分、信任、长期往来
│  │  ├─ 半熟关系：讲介绍人、礼数、试探
│  │  ├─ 陌生关系：讲规则、利益、边界
│  │  └─ 权力关系：讲位置、资历、资源控制
│  ├─ 圈子
│  │  ├─ 家庭圈、同乡圈、同学圈、单位圈、行业圈
│  │  └─ 圈内讲信任，圈外讲证明
│  └─ 分寸
│     ├─ 亲近不越界
│     ├─ 热情不压迫
│     ├─ 帮忙不包办
│     └─ 拒绝不伤人
│
├─ 3. 交往原则
│  ├─ 先尊重后表达：先承认对方位置，再提出自己意见
│  ├─ 先听懂潜台词：很多话不是字面意思
│  ├─ 先铺垫后请求：重大请求通常不宜突然开口
│  ├─ 先维护关系再解决问题：尤其在家庭、单位、熟人社会中
│  ├─ 能私下说的不公开说：批评、拒绝、利益冲突尤其如此
│  ├─ 让别人有台阶：哪怕不同意，也给对方保留解释空间
│  └─ 不把话说死：留余地，方便后续调整
│
├─ 4. 语言与潜台词
│  ├─ 委婉表达
│  │  ├─ “我考虑一下”可能是婉拒
│  │  ├─ “有机会再说”多半是不承诺
│  │  ├─ “不太方便”通常是明确拒绝
│  │  └─ “你看着办”可能是真授权，也可能是考验判断
│  ├─ 客套话
│  │  ├─ “改天请你吃饭”不一定是具体邀约
│  │  ├─ “来都来了”常用于缓和或劝说
│  │  └─ “别客气”不等于完全没有边界
│  ├─ 赞美
│  │  ├─ 宜具体，不宜过度
│  │  ├─ 夸人可夸努力、眼光、能力、孩子、家庭
│  │  └─ 避免让人觉得虚伪或有所图
│  └─ 批评
│     ├─ 私下说
│     ├─ 先肯定再指出问题
│     ├─ 对事不对人
│     └─ 给解决方案而不是只否定
│
├─ 5. 礼尚往来
│  ├─ 礼物
│  │  ├─ 目的：表达心意、感谢、维系关系
│  │  ├─ 原则：合适比贵重重要
│  │  ├─ 禁忌：过重会变成人情压力，甚至触碰纪律法律
│  │  └─ 场景：节日、探病、乔迁、婚丧、生育、拜访
│  ├─ 请客吃饭
│  │  ├─ 饭局是信息、情感、关系的综合场
│  │  ├─ 主人负责照顾氛围和座次
│  │  ├─ 客人要懂感谢、回应、不过度挑剔
│  │  └─ 买单常体现关系定位和人情流向
│  ├─ 回礼
│  │  ├─ 收到帮助要表达感谢
│  │  ├─ 不能马上还，也要记住
│  │  └─ 回礼讲时机、轻重和对方需要
│  └─ 红白喜事
│     ├─ 婚礼、生育、升学、乔迁：重在祝贺
│     ├─ 丧事、探病、困难：重在慰问
│     └─ 随礼金额常受亲疏、地区、圈层影响
│
├─ 6. 家庭与亲戚
│  ├─ 长幼秩序：尊重长辈、照顾晚辈，是传统伦理核心
│  ├─ 孝与边界：孝顺不等于无条件服从
│  ├─ 亲戚往来：节日问候、婚丧到场、困难帮衬
│  ├─ 家族面子：个人选择常被放进家庭评价体系
│  ├─ 代际冲突：婚恋、职业、育儿、养老最常见
│  └─ 成熟做法：尊重情感，明确边界，重大决定自己负责
│
├─ 7. 职场世故
│  ├─ 与上级
│  │  ├─ 及时汇报，别让上级意外
│  │  ├─ 提问题时带方案
│  │  ├─ 公开场合维护基本尊重
│  │  └─ 不越级、不抢功、不甩锅
│  ├─ 与同事
│  │  ├─ 合作留痕，利益说清
│  │  ├─ 帮忙要有边界
│  │  ├─ 不轻易卷入派系和八卦
│  │  └─ 功劳可分享，责任要明确
│  ├─ 与下属
│  │  ├─ 给方向、资源和反馈
│  │  ├─ 批评私下，表扬公开
│  │  ├─ 不用情绪管理团队
│  │  └─ 保护下属合理利益
│  └─ 职场饭局
│     ├─ 观察座次、话题、敬酒、发言顺序
│     ├─ 不乱承诺、不乱评价人
│     └─ 酒桌文化可礼貌参与，也应守住健康和法律边界
│
├─ 8. 熟人社会办事逻辑
│  ├─ 找人帮忙
│  │  ├─ 先判断关系够不够
│  │  ├─ 说明事情、难度、风险
│  │  ├─ 给对方拒绝空间
│  │  └─ 成与不成都要感谢
│  ├─ 被人求助
│  │  ├─ 能帮多少说多少
│  │  ├─ 不轻易超出能力承诺
│  │  ├─ 涉及违法违规必须拒绝
│  │  └─ 拒绝时给理由和替代建议
│  └─ 中间人
│     ├─ 介绍人承担信用背书
│     ├─ 双方都要给介绍人面子
│     └─ 事情失败也要避免让中间人难堪
│
├─ 9. 场景规则
│  ├─ 拜访：提前约、带小礼、不过久留、离开后致谢
│  ├─ 聚餐：照顾他人忌口，别只顾自己，别强迫喝酒
│  ├─ 微信：重要事别只发一句，催人要留余地
│  ├─ 节日：问候重在心意，群发可用但别太敷衍
│  ├─ 探病：少问隐私，少讲负面，多给实际帮助
│  ├─ 婚礼：准时、随礼、祝福，不抢主人风头
│  └─ 丧事：庄重、少话、帮忙比安慰话更有用
│
├─ 10. 分寸与边界
│  ├─ 关系再好，也不应默认占用别人时间、金钱、资源
│  ├─ 玩笑要看对象、场合、身份差异
│  ├─ 不打听收入、婚育、病情、家庭矛盾等隐私
│  ├─ 不用“为你好”压迫别人
│  ├─ 不把帮忙变成道德绑架
│  └─ 真正高级的人情世故，是让别人舒服，也让自己不委屈
│
└─ 11. 现代变化
   ├─ 年轻人更重视边界、平等、效率
   ├─ 城市职场更依赖契约和规则
   ├─ 传统家庭仍重视礼数和情分
   ├─ 互联网沟通削弱了部分客套，也制造了新的误会
   └─ 最稳妥的能力：懂传统规则，但不被旧规则绑架
```
